# Demos

We accept demos, listen to everything we get sent, and always reply.

We're focused on releasing peoples first physical (e.g. vinyl) release.

---

### Guidelines:

- Music should have some live performance element (e.g. no 100% sequenced computer music).
- We're more interested if you haven't released physical media before.
- We prefer [<u>untitled.stream</u>](https://untitled.stream) for demos.
- Send 4-8 tracks.
